see

An alternative to Python’s dir function. Easy to type; easy to read! For humans only.

Version 1.0.1, 17 October 2010

Install

To download and install automatically, simply run:

# easy_install -U see

Alternatively, to install from source:

# python setup.py install

If you don’t have root privileges, these commands may be run with --prefix="$HOME" appended to install to your home directory.

Usage

>>> from see import see
>>>
>>> see()
    os    see()    sys
>>> see([])
    []           in           +            +=           *            *=
    <            <=           ==           !=           >            >=
    hash()       help()       iter()       len()        repr()
    reversed()   str()        .append()    .count()     .extend()    .index()
    .insert()    .pop()       .remove()    .reverse()   .sort()
>>> see(str, '.is*')
    .isalnum()    .isalpha()    .isdigit()    .islower()    .isspace()    .istitle()
    .isupper()

If you don’t want to type from see import see every time you run the interpreter, you can make a Python startup file containing this line. Information on how to do this is in the readme file.

Download

The latest source code can be found on GitHub. You can clone the project with Git by running:

$ git clone git://github.com/inky/see
